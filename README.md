# 8-Puzzle Solver

A web-based application that solves the classic 8-puzzle problem using AI search algorithms with real-time visualization.

## Features
- Solve 8-puzzle problems using A* and Best-First Search
- Compare Manhattan Distance and Misplaced Tiles heuristics
- Interactive puzzle board with step-by-step solutions
- Custom puzzle input and random puzzle generation
- Performance metrics and solution visualization

## Quick Start

### Backend Setup
```bash
cd backend
pip install -r requirements.txt
python app.py
```

### Frontend Setup
```bash
cd frontend/puzzle-ui
npm install
npm start
```
